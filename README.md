## Installation

### Clone

```bash
https://github.com/jessicsw/Groove.git
```

### Setup

Install project dependencies:

```bash
npm i
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see Groove in action.
